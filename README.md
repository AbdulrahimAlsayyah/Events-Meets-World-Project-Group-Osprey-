## About the Project
Events Meets World is a business platform that is designed to help individuals with event planning by connecting vendors with clients through a dynamic interface. The system allows users to manage, create and execute events by having the exposure to vendors, communication and securing payment right there and then. The team uses a strucutured and agile development approach to complete this project.
## Objectives 
1. Centralised platform to use
2. UX/UI high standard through navigation
3. Communciation is secured between the Vendors and the Client
4. GDPR is ensured
